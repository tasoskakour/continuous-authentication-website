Keystroke dynamics, keystroke biometrics or typing dynamics, is the detailed timing information which describes exactly when each key was pressed and when it was released as a person is typing at a computer keyboard (more here).


The present site, uses in addition to the regular username-password authentication, a continuous authentication mechanism based, exactly on keystroke dynamics.
Specifically, after the first level-authentication of username-password provided at the login procedure, the authentication continues, by recording the keystroke timings and patterns of the user and the system either allows the user to continue browsing the website or classify the user as a malicious one and locks him out of the system.



Of course, in order for this mechanism to work, the system needs enough keystroke patterns and timings of the user so it can build his keystroke dynamics profile, and eventually when the data is enough the system will apply the mechanism described above.


So in order to collect enough keystroke dynamics data of yours, we offer you some simple use case scenarios simulations of the real world where a user need to type something that is at some times very sensitive (like his bank-account information) or not so sensitive (like a writing of a friendly e-mail).

Also by visiting your profile you can see some cool stuff of your keystroke dynamics profile (like your timings).
